# AGENTS.md — `simonsangla/lab`

Static "One Day One App" showcase. Zero build step. The homepage `index.html` is the hand-authored **Hire Simon** freelance landing page (sells Simon, links a curated few apps). The full app gallery is `gallery.html`, regenerated by GitHub Actions on every push to `apps/**`. Vercel serves the repo root.

## How to ship a new app

1. Create `apps/YYYY-MM-DD-app-name.html` — standalone HTML, all CSS/JS inline.
2. Required meta tags in `<head>`:

   ```html
   <meta name="app-name" content="...">
   <meta name="app-description" content="...">
   <meta name="app-tags" content="tag1,tag2">
   ```

3. Commit on a feature branch → push → open PR → merge. The post-merge push to `main` triggers `.github/workflows/gen-index.yml`, which regenerates `gallery.html` with the new card.

Apps' "back to lab" link points to `/gallery.html` (the full list); the homepage (`index.html`) is reached via the gallery masthead. To keep an app out of the public gallery, add `<meta name="app-visibility" content="private">` to its `<head>` (currently set on `2026-06-19-ai-money-map.html`); the file stays in `apps/`, still deploys, and remains directly linkable.

## Conventions

- Mobile-first. Brand identity (App Store-style light theme, Basque green `#009A44`) is the reusable component [assets/lab-theme.css](assets/lab-theme.css): apps link it **after** their inline `<style>` (`<link rel="stylesheet" href="/assets/lab-theme.css">`) and it re-maps the shared `:root` tokens (`--bg`, `--surface`, `--accent`, …) plus `.lab-*` utility classes. Keep the inline dark tokens in each app — they are the standalone/file:// fallback when the asset can't load. Text-bearing greens must use `--lab-green-text`/`--lab-green-dark` (AA on light surfaces), not raw `#009A44`. The app gallery (`gallery.html`) is generated by `scripts/gen-index.mjs` in the same identity; the homepage `index.html` (Hire Simon) is hand-authored and edited directly.
- No external runtime deps. Google Fonts via `<link>` only.
- App files must be openable standalone (no shared CSS/JS outside the file).
- Filename `YYYY-MM-DD-` prefix is the sort key (newest first).

## Generator

[scripts/gen-index.mjs](scripts/gen-index.mjs) — Node ESM, stdlib only. Run `npm run gen` (or `node scripts/gen-index.mjs`) for a local preview. Idempotent.

## Factory

The repeatable pipeline (idea → spec → app → gates → deploy → evidence) is
documented in [docs/APP_FACTORY.md](docs/APP_FACTORY.md); the reuse map and
pattern catalog in [REUSE.md](REUSE.md). Before pushing an app run the gates:
`npm run lint` (meta/filename + inline-JS syntax), `npm run smoke` (DOM-stub
harness, all apps must load/render with no errors), `npm run gen` (`gallery.html`
must be regenerated and committed — CI fails stale). On PRs, `verify-apps.yml`
re-runs the gates and screenshots the Vercel preview (console errors fail).
